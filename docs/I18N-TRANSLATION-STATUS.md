## Comment traduire un composant

### 1. Importer useTranslations
```typescript

```

### 2. Initialiser le hook
```typescript

  const t = useTranslations("namespace");
  // ...
}
```

### 3. Remplacer les textes hardcodÃ©s
```typescript

// AprÃ¨s

```

### 4. Ajouter les clÃ©s dans les fichiers JSON
```json
// messages/fr.json
{
  "namespace": {
    "actions": {
      "save": "Enregistrer"
    }
  }
}

// messages/en.json
{
  "namespace": {
    "actions": {
      "save": "Save"
    }
  }
}
```

### 5. Interpolation de variables
```typescript

// JSON
"greeting": "Bonjour {name} !"
```
